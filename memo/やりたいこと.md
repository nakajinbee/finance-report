# 企業会計情報 WEB アプリ 開発メモ

## 概要
EDINET API を使って企業の会計情報をさっと確認できる WEB アプリを作る。

## 進め方（アジャイル 2サイクル）

### サイクル 1：コア機能
1. **要件定義** — 最低限動くものの要件を決める
2. **設計** — 技術スタック・データフロー・画面設計
3. **開発** — 実装
4. **テスト** — 動作確認・バグ修正

### サイクル 2：追加機能
1. **追加要件定義** — サイクル1の使用感をもとに追加・改善点を決める
2. **設計** — 追加機能の設計
3. **開発** — 実装
4. **テスト** — 動作確認・バグ修正

---

## サイクル 1 要件
→ docs/requirements/cycle1_requirements.md 参照

### 概要
- 対象：リクルートホールディングス（証券コード：6098、IFRS採用）
- FR-01：EDINET から有価証券報告書を直近5期分取得
- FR-02：売上高・営業利益・純利益・総資産・負債をグラフ表示（指標切り替え）
- FR-03：エラー・データなし時のメッセージ表示
