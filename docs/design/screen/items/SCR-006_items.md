# SCR-006 比較結果画面 画面項目定義

親画面：[../SCR-006_comparison_result.md](../SCR-006_comparison_result.md)

各画面項目が、どのAPI（[api_list.md](../../api/api_list.md)参照）の、どのレスポンス項目から来ているかを定義する。

---

## 項目一覧

| 項目ID | 画面項目名 | 表示位置 | 参照元API | 参照元レスポンス項目 | 表示形式・整形ルール | 備考 |
|---|---|---|---|---|---|---|
| SCR-006-01 | 戻るリンク | 画面左上 | なし（画面遷移のみ） | ― | 押下でSCR-005へ遷移 | |
| SCR-006-02 | 見出し「比較結果（{企業数}社）」 | 画面上部 | なし（フロント内処理） | `codes`の企業数（取得成功分） | | |
| SCR-006-03 | B/S：企業別グラフ・表 | B/Sセクション | API-COM-002 | 各企業の`data[]`の最新期レコードから`total_assets`/`total_liabilities`/`equity` | グループ棒グラフ（x軸＝企業名、系列＝3指標）＋表 | SCR-003機能5と同じ整形ルール |
| SCR-006-04 | P/L：企業別グラフ・表 | P/Lセクション | API-COM-002 | 各企業の最新期レコードから`revenue`/`operating_profit`/`ordinary_profit`/`net_profit` | 機能SCR-006-03と同様 | |
| SCR-006-05 | キャッシュフロー：企業別グラフ・表 | CFセクション | API-COM-003 | 各企業の最新期`CashFlowRecord` | 機能SCR-006-03と同様 | |
| SCR-006-06 | 財務分析指標：企業別グラフ・表×4カテゴリ | 財務分析指標エリア | API-COM-005 | 各企業の最新期`RatioRecord`の該当フィールド | カテゴリごとにグループ棒グラフ（x軸＝企業名）＋表 | SCR-003機能11と同じカテゴリ分類 |
| SCR-006-07 | グラフ・表の企業名クリック | 各グラフ・表 | ― | `code` | クリックで`/companies/{code}`へ遷移 | |
| SCR-006-08 | 一部失敗の警告表示 | 画面上部 | API-COM-002/003/005 | 呼び出し失敗した企業のリスト | 「一部の企業のデータ取得に失敗しました（{企業名}）」 | 失敗企業はグラフ・表から除外 |
| SCR-006-09 | 空状態の案内メッセージ | 画面中央 | なし（フロント内処理） | `codes`が空、または全企業取得失敗 | 「比較する企業が選択されていません」＋SCR-005へのリンク | |

## 画面状態とAPIレスポンスの対応

| 画面状態 | 判定に使うレスポンス項目 |
|---|---|
| ローディング | API-COM-002/003/005 呼び出し中 |
| 正常 | `codes`のうち1件以上の企業でAPI呼び出しが成功 |
| 空 | `codes`が空、または全企業で失敗 |
| 一部失敗 | 一部の企業でAPI-COM-002/003/005が`404`/`500`を返却 |
