# SCR-003 企業詳細画面 画面項目定義

親画面：[../SCR-003_company_detail.md](../SCR-003_company_detail.md)

各画面項目が、どのAPI（[api_list.md](../../api/api_list.md)参照）の、どのレスポンス項目から来ているかを定義する。

---

## 項目一覧

| 項目ID | 画面項目名 | 表示位置 | 参照元API | 参照元レスポンス項目 | 表示形式・整形ルール | 備考 |
|---|---|---|---|---|---|---|
| SCR-003-01 | 戻るリンク | 画面左上 | なし（画面遷移のみ） | ― | 押下でSCR-002へ遷移。APIは呼ばない | |
| SCR-003-02 | 企業ヘッダー：企業名 | 画面上部 | API-COM-002<br>`GET /api/companies/{code}/financials` | `CompanyFinancials.company.name` | そのまま表示 | パスパラメータ`code`はSCR-002-08（企業カードクリック）から引き継ぐ |
| SCR-003-03 | 企業ヘッダー：会計基準 | 画面上部 | API-COM-002 | `CompanyFinancials.company.accounting_standard` | 「会計基準：IFRS」の形式で表示 | |
| SCR-003-04 | 表示期間選択（3期/5期/10期） | ヘッダー下 | なし（フロント内処理） | ― | API-COM-002で取得済みの `data[]` を `period_end` 降順に並べ、先頭N件のみグラフに渡す | API再呼び出しなし。データが選択期数より少ない場合は存在する分のみ表示 |
| SCR-003-05 | 指標トグルボタン（売上高/営業利益/純利益/総資産/負債合計） | 表示期間選択の下 | なし（フロント内処理） | ― | ON/OFFはフロントの表示状態のみで管理し、グラフに渡すデータ系列を絞り込む | データ自体はSCR-003-06と同じ`data[]`を参照 |
| SCR-003-06 | グラフ：横軸（会計年度） | グラフ下部 | API-COM-002 | `data[].fiscal_year` | そのまま表示（例：「2023年3月期」）。表示期間選択（SCR-003-04）に連動して絞り込み | |
| SCR-003-07 | グラフ：売上高の棒 | グラフ本体 | API-COM-002 | `data[].revenue` | 円→億円に変換して表示（÷100,000,000） | `null`の場合はその期の棒を描画しない |
| SCR-003-08 | グラフ：営業利益の棒 | グラフ本体 | API-COM-002 | `data[].operating_profit` | 同上 | 同上 |
| SCR-003-09 | グラフ：純利益の棒 | グラフ本体 | API-COM-002 | `data[].net_profit` | 同上 | 同上 |
| SCR-003-10 | グラフ：総資産の棒 | グラフ本体 | API-COM-002 | `data[].total_assets` | 同上 | 同上 |
| SCR-003-11 | グラフ：負債合計の棒 | グラフ本体 | API-COM-002 | `data[].total_liabilities` | 同上 | 同上 |
| SCR-003-12 | 凡例 | グラフ下部 | なし（フロント内処理） | ― | SCR-003-05でONになっている指標のみ、カラー定義（画面定義書参照）に従って表示 | |
| SCR-003-13 | ツールチップ：指標名と金額 | 棒ホバー時 | API-COM-002 | ホバー中の`data[]`要素における、ONの各指標フィールド（SCR-003-07〜11と同じ） | 1兆円以上「X.X兆円」、1兆円未満「X,XXX億円」に整形 | 該当フィールドが`null`の場合は「データなし」と表示 |

## 画面状態とAPIレスポンスの対応

| 画面状態 | 判定に使うレスポンス項目 |
|---|---|
| ローディング | API-COM-002 呼び出し中（応答待ち） |
| 正常 | API-COM-002 が `200` を返し `data[]` が1件以上 |
| 全指標OFF | SCR-003-05の全トグルがOFF（フロント内状態、APIとは無関係） |
| データなし | API-COM-002 の `data[]` 全要素で、対象5指標（revenue/operating_profit/net_profit/total_assets/total_liabilities）がすべて `null` |
| エラー | API-COM-002 が `404`（企業なし）または `500` エラーを返却（`Error`スキーマ） |
