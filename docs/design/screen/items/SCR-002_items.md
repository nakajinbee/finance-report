# SCR-002 企業一覧画面 画面項目定義

親画面：[../SCR-002_company_list.md](../SCR-002_company_list.md)

各画面項目が、どのAPI（[api_list.md](../../api/api_list.md)参照）の、どのレスポンス項目から来ているかを定義する。

---

## 項目一覧

| 項目ID | 画面項目名 | 表示位置 | 参照元API | 参照元レスポンス項目 | 表示形式・整形ルール | 備考 |
|---|---|---|---|---|---|---|
| SCR-002-01 | 検索バー | 画面上部 | なし（フロント内処理） | ― | API-COM-001で取得済みの一覧をフロントで絞り込む。API再呼び出しなし | 検索対象はSCR-002-04の企業名・証券コード |
| SCR-002-02 | 業種絞り込みセレクト（サイクル14追加） | 検索バーの隣 | API-COM-001 | `Company.sector`（重複除去） | 取得済み一覧から`sector`の重複なしリストを五十音順に生成し先頭に「すべて」を追加。`null`の企業は選択肢生成から除く | 検索バーとAND条件で絞り込む |
| SCR-002-03 | 並び順セレクト（サイクル14追加） | 業種絞り込みの隣 | なし（フロント内処理） | ― | 「証券コード昇順」「証券コード降順」「企業名昇順」「企業名降順」の4択。初期値は証券コード昇順 | 絞り込み後の結果に適用 |
| SCR-002-04 | 件数表示「{全件数}件中{表示件数}件表示」 | 絞り込み欄の下 | API-COM-001<br>`GET /api/companies` | 全件数＝レスポンス配列（`Company[]`）の要素数、表示件数＝絞り込み後の件数 | 「3件中3件表示」の形式で表示。絞り込み条件（キーワードまたは業種）がない場合は非表示 | 表示件数はフロント側の絞り込み結果を都度カウント |
| SCR-002-05 | 案内メッセージ（サイクル14追加） | 件数表示の位置 | なし（フロント内処理） | ― | 「企業名・証券コードで検索するか、業種を選択してください」 | 絞り込み条件（キーワードまたは業種）がない場合のみ表示。件数表示・カード一覧とは排他 |
| SCR-002-06 | 企業カード一覧（コンテナ） | 画面中央 | API-COM-001 | レスポンス配列 `Company[]` | SCR-002-03の並び順でソートして描画 | 初回のみAPI呼び出し。以降の検索・絞り込み・並び替えはフロントで処理。絞り込み条件がない場合は非表示 |
| SCR-002-07 | 企業カード：企業名 | カード内1行目 | API-COM-001 | `Company.name` | そのまま表示（例：「リクルートホールディングス」） | |
| SCR-002-08 | 企業カード：証券コード | カード内2行目 | API-COM-001 | `Company.code` | そのまま表示（例：「6098」） | |
| SCR-002-09 | 企業カード：業種 | カード内2行目 | API-COM-001 | `Company.sector` | そのまま表示（例：「サービス業」）。`null`の場合は「業種不明」と表示 | |
| SCR-002-10 | 企業カード：会計基準 | カード内2行目 | API-COM-001 | `Company.accounting_standard` | そのまま表示（例：「IFRS」） | |
| SCR-002-11 | 企業カード：クリック時の遷移先 | カードクリック | API-COM-001 | `Company.code` | `/companies/{code}` へ遷移し、SCR-003で`code`をパスパラメータとして使用 | SCR-003側は`API-COM-002`の`code`パラメータに引き継ぐ |
| SCR-002-12 | データ追加取得ボタン | 画面下部 | なし（画面遷移のみ） | ― | 押下でSCR-001へ遷移。APIは呼ばない | |

## 画面状態とAPIレスポンスの対応

| 画面状態 | 判定に使うレスポンス項目 |
|---|---|
| ローディング | API-COM-001 呼び出し中（応答待ち） |
| 正常（データなし） | API-COM-001 のレスポンス配列が0件 |
| 未絞り込み（サイクル14で追加） | API-COM-001 のレスポンス配列が1件以上、かつキーワード未入力・業種未選択 |
| 正常（絞り込み結果あり） | キーワードまたは業種の絞り込み条件があり、結果が1件以上 |
| 絞り込み結果0件 | キーワードまたは業種の絞り込み条件があるが、結果が0件 |
| エラー | API-COM-001 が `500` エラーを返却（`Error`スキーマ） |
