# SCR-005 比較企業選択画面 画面項目定義

親画面：[../SCR-005_comparison_select.md](../SCR-005_comparison_select.md)

各画面項目が、どのAPI（[api_list.md](../../api/api_list.md)参照）の、どのレスポンス項目から来ているかを定義する。

---

## 項目一覧

| 項目ID | 画面項目名 | 表示位置 | 参照元API | 参照元レスポンス項目 | 表示形式・整形ルール | 備考 |
|---|---|---|---|---|---|---|
| SCR-005-01 | 検索バー | 画面上部 | なし（フロント内処理） | ― | SCR-002-01と同じ | |
| SCR-005-02 | 業種絞り込みセレクト | 検索バーの隣 | API-COM-001 | `Company.sector`（重複除去） | SCR-002-02と同じ | |
| SCR-005-03 | 並び順セレクト | 業種絞り込みの隣 | なし（フロント内処理） | ― | SCR-002-03と同じ | |
| SCR-005-04 | 選択中の企業リスト（コンテナ） | 絞り込み欄の下 | なし（フロント内処理） | ― | 選択した順に表示。「選択中の企業（{件数}）」の見出し | 選択0件の場合は非表示 |
| SCR-005-05 | 選択中の企業：除外ボタン | 選択中リスト各行 | なし（フロント内処理） | ― | 押下でリストから除外し、企業一覧側の選択状態も解除 | |
| SCR-005-06 | 比較するボタン | 選択中リストの下 | なし（画面遷移のみ） | ― | 選択中の企業コードをカンマ区切りで`/compare/result?codes=`に渡して遷移 | 選択0件の場合は無効化 |
| SCR-005-07 | 案内メッセージ（未絞り込み時） | 企業一覧の位置 | なし（フロント内処理） | ― | SCR-002-05と同じ文言 | |
| SCR-005-08 | 企業一覧（コンテナ） | 画面中央 | API-COM-001 | レスポンス配列 `Company[]` | 並び順セレクトの選択に従いソート | 絞り込み条件がある場合のみ表示 |
| SCR-005-09 | 企業カード：選択状態の表示 | 各カード | なし（フロント内処理） | ― | 未選択「+ {企業名}」・選択済み「✓ {企業名}（選択済み）」＋背景色変更 | SCR-002のカードと見た目を変える（クリックで詳細へ遷移しないことを示す） |
| SCR-005-10 | 企業カード：クリック時の処理 | 各カード | なし（フロント内処理） | ― | クリックで選択中リストへの追加／除外をトグル | SCR-002と異なりSCR-003へは遷移しない |
| SCR-005-11 | 企業カード：業種・会計基準等の表示項目 | 各カード | API-COM-001 | `Company.sector` / `Company.accounting_standard` | SCR-002-09・SCR-002-10と同じ | |

## 画面状態とAPIレスポンスの対応

| 画面状態 | 判定に使うレスポンス項目 |
|---|---|
| ローディング | API-COM-001 呼び出し中 |
| 正常（データなし） | API-COM-001 のレスポンス配列が0件 |
| 未絞り込み | API-COM-001 のレスポンス配列が1件以上、かつキーワード未入力・業種未選択 |
| 正常（絞り込み結果あり） | キーワードまたは業種の絞り込み条件があり、結果が1件以上 |
| 絞り込み結果0件 | キーワードまたは業種の絞り込み条件があるが、結果が0件 |
| エラー | API-COM-001 が `500` エラーを返却（`Error`スキーマ） |
