# SCR-005 比較企業選択画面

> 画面項目とAPIレスポンスの対応は [items/SCR-005_items.md](items/SCR-005_items.md) を参照。

## 基本情報

| 項目 | 内容 |
|------|------|
| 画面ID | SCR-005 |
| 画面名 | 比較企業選択画面 |
| パス | `/compare` |
| 呼び出し元 | 共通ヘッダーの「企業比較」リンク |
| 関連サイクル | サイクル15 FR-65（UC-1-2-1：同業他社と指標を横並び比較する） |

**設計の経緯（2026-07-24）**：当初はSCR-003（企業詳細画面）に「比較に追加」
操作を設け、1社ずつ比較対象を蓄積する案で設計を進めたが、ユーザーから
「企業詳細画面を経由して1社ずつ追加するのは、比較相手を探しにくい」との
指摘があり、企業一覧と同じ検索UXを持つ専用の選択画面（本画面）に変更した。

## 機能定義

### 1. 検索バー・業種絞り込み・並び順（SCR-002と同じ仕様を流用）

| 項目 | 内容 |
|------|------|
| 検索バー | SCR-002と同じ（企業名・証券コードの部分一致、リアルタイム絞り込み） |
| 業種絞り込み | SCR-002と同じ（取得済み企業一覧から`sector`の重複除去、五十音順、
  `null`除外、「すべて」＋一覧） |
| 並び順 | SCR-002と同じ（証券コード昇順・降順、企業名昇順・降順） |
| 未絞り込み時の表示 | SCR-002と同じ方針（キーワード未入力・業種未選択の場合は
  企業一覧を表示せず案内メッセージのみ） |

### 2. 企業一覧（SCR-002とは挙動が異なる）

| 項目 | 内容 |
|------|------|
| 表示条件 | 絞り込み条件（キーワードまたは業種）がある場合のみ |
| クリック時の処理 | **SCR-002と異なり、企業詳細画面へは遷移しない**。クリックした
  企業を「選択中の企業リスト」（機能3）に追加する（トグル。選択済みの企業を
  再度クリックすると選択解除する） |
| 選択状態の表示 | 未選択のカードは「+ {企業名}」、選択済みのカードは
  「✓ {企業名}（選択済み）」と表示し、背景色も変える（`bg-brand-tint`）。
  SCR-002のカード（クリックで詳細へ遷移）とは明確に見た目を変え、
  「クリックで選択される一覧である」ことを示す |
| 表示項目 | SCR-002と同じ（企業名・証券コード・業種・会計基準） |

### 3. 選択中の企業リスト

| 項目 | 内容 |
|------|------|
| 表示位置 | 検索バー・業種絞り込みの下、企業一覧より上 |
| 表示内容 | 選択した順に企業名・証券コードを並べる。「選択中の企業（{件数}）」の
  見出しを付ける |
| 除外ボタン | 各企業に除外ボタンを設ける。押下でリストから外し、企業一覧側の
  選択状態（機能2）も解除する |
| 保持方法 | 画面内のクライアント状態（`useState`等）のみで保持する。他画面に
  遷移すると破棄される（バックエンド・`localStorage`への保存はしない） |
| 表示条件 | 選択中の企業が1件以上の場合のみ表示。0件の場合は非表示 |
| 選択数の上限 | 設けない（SCR-006比較結果画面は横スクロール前提で対応する） |

### 4. 比較するボタン

| 項目 | 内容 |
|------|------|
| 表示条件 | 常時表示 |
| 活性化条件 | 選択中の企業が1件以上ある場合のみ活性化。0件の場合は無効化
  （`opacity-50` `cursor-not-allowed`） |
| 押下時の処理 | 選択中の企業コードをカンマ区切りにして、SCR-006（比較結果画面）へ
  `/compare/result?codes=1301,7203`の形式で遷移する |

## 画面状態

| 状態 | 説明 | 表示内容 |
|------|------|----------|
| ローディング | API-COM-001 呼び出し中 | 「読み込み中...」 |
| 正常（データなし） | 保存済み企業が0件 | SCR-002と同じ「データがありません」表示 |
| 未絞り込み | キーワード未入力・業種未選択 | 検索バー・絞り込み・選択中リストは表示するが、
  企業一覧は非表示（案内メッセージ） |
| 正常（絞り込み結果あり） | 絞り込み条件があり結果が1件以上 | 企業一覧を表示 |
| 絞り込み結果0件 | 絞り込み条件があるが結果が0件 | SCR-002と同じ「見つかりませんでした」表示 |
| エラー | API-COM-001 がエラーを返却 | SCR-002と同じエラー表示 |
