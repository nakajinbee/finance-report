# SCR-003 企業詳細画面

> 画面項目とAPIレスポンスの対応は [items/SCR-003_items.md](items/SCR-003_items.md) を参照。

## 基本情報

| 項目 | 内容 |
|------|------|
| 画面ID | SCR-003 |
| 画面名 | 企業詳細画面 |
| パス | `/companies/:code` |
| 呼び出し元 | SCR-002（企業一覧画面）の企業カードクリック |

サイクル2で期間選択方式の変更（FR-12：3期/5期/10期プリセット→開始年度〜終了年度）と、
キャッシュフロー計算書の表示（FR-13）を追加した。サイクル3で財務分析指標
セクション（FR-23〜26）を追加した。**2026-07-23（サイクル13）で本ドキュメントを
実装に合わせて全面是正**：B/S・P/Lの2セクション分離、指標の追加（経常利益・
自己資本）、各セクションへの表の追加、財務分析指標セクションのグラフ表示を
反映していなかった（実装済みの内容がドキュメントに反映されないまま複数サイクルが
経過していた）。あわせて定性情報セクション（FR-58）を新設し、SCR-004（生データ確認画面、
ユースケースに紐づかない開発者向け機能と判断）への遷移ボタンを削除した。

## 機能定義

### 1. 戻るリンク

| 項目 | 内容 |
|------|------|
| 表示条件 | 常時表示 |
| 押下時の処理 | SCR-002（企業一覧画面）へ遷移 |

### 2. 企業ヘッダー

| 項目 | 内容 |
|------|------|
| 表示内容 | 企業名・会計基準 |
| データ取得元 | `GET /api/companies/{code}/financials` のレスポンスから取得 |

### 3. 表示期間選択

| 項目 | 内容 |
|------|------|
| 選択方式 | 開始年度・終了年度をそれぞれセレクタで指定 |
| 選択肢の範囲 | その企業についてDBに保存済みの年度のみを選択肢に出す |
| 初期選択 | DBに保存済みの最も古い年度〜最も新しい年度（＝全期間表示） |
| 選択時の処理 | `GET /api/companies/{code}/financials?from_year=&to_year=`・
  `GET /api/companies/{code}/cashflow?from_year=&to_year=`・
  `GET /api/companies/{code}/ratios?from_year=&to_year=`を再呼び出しする |
| 連動範囲 | 機能5（B/S）・機能6（P/L）・機能7（CF）・機能8（財務分析指標）すべてに
  連動する。**機能9（事業概要・リスク）は連動しない**（独立した単一年度セレクタを持つ、
  サイクル13 FR-58） |
| データ不足時の挙動 | 指定範囲内にデータが存在しない年度は「データなし」として扱う |

### 4. 指標トグルボタン（各セクション共通の仕組み）

機能5（B/S）・機能6（P/L）は、それぞれ独立した指標トグル（チェックボックス形式、
複数同時選択可、初期状態は全指標ON）を持つ。ONの指標のみグラフ・表に表示する
（API再呼び出しなし、フロント内の表示状態のみで制御）。機能8（財務分析指標）も
同様の仕組みだが、初期状態が異なる（機能8参照）。

### 5. 貸借対照表（B/S）セクション

| 項目 | 内容 |
|------|------|
| 対象指標 | 総資産・負債・自己資本（純資産）の3指標 |
| 表示形式 | グループ棒グラフ＋表（`FinancialMetricSection`。グラフはグループ棒グラフ、
  横軸は会計年度、縦軸は億円）＋指標トグル |
| データ取得元 | `GET /api/companies/{code}/financials?from_year=&to_year=` |
| ツールチップ | 棒にカーソルを当てるとON指標の「{指標名}：{金額}」を表示
  （1兆円以上「X.X兆円」、1兆円未満「X,XXX億円」） |
| データなしの場合 | 全指標OFF時「指標を1つ以上選択してください」。値が全期間`null`の場合
  「表示できるデータがありません」。個別セルは「データなし」 |

### 6. 損益計算書（P/L）セクション

| 項目 | 内容 |
|------|------|
| 対象指標 | 売上高・営業利益・経常利益・純利益の4指標 |
| 表示形式 | 機能5と同じ仕組み（`FinancialMetricSection`を指標定義だけ変えて再利用） |
| 備考 | 経常利益はJapan GAAP特有の概念のため、IFRS/US GAAP企業では常にデータなしになる |

### 7. キャッシュフロー計算書

| 項目 | 内容 |
|------|------|
| 表示位置 | B/S・P/Lセクションの下（別セクション、CF単独） |
| 表示形式 | グラフ（グループ棒グラフ、営業CF/投資CF/財務CFの3系列）＋表の両方 |
| データ取得元 | `GET /api/companies/{code}/cashflow?from_year=&to_year=` |
| 金額の表示形式 | 機能5と同じ整形ルール |
| データなしの場合 | 表側は該当セルに「データなし」。グラフ側は`null`の期の棒を描画しない |

### 8. 財務分析指標セクション

| 項目 | 内容 |
|------|------|
| 表示位置 | キャッシュフロー計算書の下、別セクション |
| カテゴリ構成 | 収益性・効率性・安全性・投資指標の4カテゴリ（`RatioCategorySection`を
  カテゴリごとに4回使用） |
| 表示形式 | カテゴリごとにグラフ（グループ棒グラフ）＋表＋トグルを表示する（サイクル3時点の
  「表形式のみ」から、サイクル4以降の実装でグラフ表示が追加されていた） |
| 内訳トグルの初期状態 | 指標本体（例：ROE）のみ選択状態、内訳（例：純利益・自己資本）は
  非選択（2026-07-22ユーザー要望：内訳は見たい人だけ表示） |
| データ取得元 | `GET /api/companies/{code}/ratios?from_year=&to_year=` |
| 数値の表示形式 | 比率は%表示、回転率は「X.X回」、EPS/PERは小数第2位までの数値表示 |
| データなしの場合 | 該当セルに「データなし」（US GAAP企業は現時点で全指標データなしの想定） |
| 開示値・計算値の優先順位 | EDINET開示値を優先し、なければ計算値で補完する |

### 9. 事業概要・リスクセクション（サイクル13新規、FR-58。データの分類上は「定性データ」）

| 項目 | 内容 |
|------|------|
| 表示位置 | 財務分析指標セクションの下、画面最下部 |
| 年度セレクタ | 単一年度を選択する専用セレクタ。機能3（表示期間選択）とは独立。
  初期値は事業概要・リスク情報が存在する最新年度 |
| 表示内容 | 事業の内容・事業等のリスク・経営者による分析（MD&A）の3項目。
  それぞれ開閉式（デフォルト閉、クリックで展開） |
| データ取得元 | `GET /api/companies/{code}/qualitative-facts?period_end=` |
| データなしの場合 | 選択年度の3項目とも空の場合「この年度の事業概要・リスク情報はありません」。
  対象企業に開示書類が1件もない場合も、セクション（パネル）自体は常に表示し、
  年度セレクタなしで「事業概要・リスク情報はありません」と表示する
  （2026-07-23訂正：当初はセクション自体を非表示にする設計だったが、
  パネルは常に表示し情報なしと明示する方針に変更） |
| 備考 | EDINETから既に取得済みのCSV内のテキストブロックを利用しており、
  追加のEDINET通信は発生しない（`document_body_ingestion`実行時に同時取得） |
| 読みやすさの調整 | CSVの値は改行を含まない連続したテキストのため、フロントエンドで
  句点「。」ごとに改行して表示する（`QualitativeFactSection`）。**既知の制約**：
  元テキストに表（例：事業部門別販売実績の推移）が含まれる場合、書類取得API
  `type=5`（CSV。XBRLデータをCSV形式に変換したもの）の時点で区切り文字が
  失われ数字が連結してしまっており、句点改行では読みやすくならない
  （2026-07-23実機確認）。`type=1`（元のXBRL／インラインXBRL＝実質HTML）を
  取得すれば表構造を保ったまま抽出できる可能性が高いが、実装規模が大きいため
  サイクル13のスコープには含めず`cycleX_backlog.md`に送る（詳細は
  [IDEA-15](../../ideas/IDEA-15_screen_display_spec_refinement.md)参照） |

## 画面状態

| 状態 | 説明 | 表示内容 |
|------|------|----------|
| ローディング | API 呼び出し中 | 「データを読み込んでいます...」 |
| 正常 | データ取得成功 | 全セクション表示 |
| 全指標OFF（B/S・P/L・財務分析指標の各セクション） | トグルを全部OFFにした | 該当セクションに「指標を1つ以上選択してください」 |
| データなし | 全指標・全期間の値がNULL | 該当セクションに「表示できるデータがありません」 |
| エラー | API エラー | 「データの取得に失敗しました。しばらくしてから再度お試しください。」 |

## 使用 API

| API | 用途 |
|-----|------|
| `GET /api/companies/{code}/financials` | 財務データ（B/S・P/L、7指標）を取得 |
| `GET /api/companies/{code}/cashflow` | キャッシュフロー（3項目）を取得 |
| `GET /api/companies/{code}/ratios` | 財務分析指標（4カテゴリ）を取得 |
| `GET /api/companies/{code}/qualitative-facts` | 定性情報（事業の内容・リスク・MD&A）を取得（サイクル13新規） |
