## 機能定義

### 1. 戻るリンク

| 項目 | 内容 |
|------|------|
| 表示条件 | 常時表示 |
| 押下時の処理 | SCR-002（企業一覧画面）へ遷移 |

### 2. 企業ヘッダー

| 項目 | 内容 |
|------|------|
| 表示内容 | 企業名・会計基準 |
| データ取得元 | `GET /api/companies/{code}/financials` のレスポンスから取得 |

### 3. 表示期間選択

| 項目 | 内容 |
|------|------|
| 選択肢 | 3期 / 5期 / 10期 |
| 初期選択 | 5期 |
| 選択時の処理 | グラフの横軸を「最新期から遡って N 期分」に絞り込む（API 再呼び出しなし） |
| データ不足時の挙動 | ボタンは選択可能。データが存在する期数分のみグラフに表示する（例：データ5期で10期を選択 → 5期分表示） |

### 4. 指標トグルボタン

| 項目 | 内容 |
|------|------|
| 表示内容 | 売上高 / 営業利益 / 純利益 / 総資産 / 負債合計 |
| 操作形式 | チェックボックス形式（複数同時選択可） |
| 初期状態 | 全指標をON |
| 選択時の処理 | ON の指標のみグラフに表示する（API 再呼び出しなし） |
| 表示形式 | ボタン内に指標名とその指標のカラーライン（━）を表示し、グラフの色と対応させる |

#### 指標のカラーリング

| 指標 | カラー |
|------|--------|
| 売上高 | Blue（#4E79A7） |
| 営業利益 | Orange（#F28E2B） |
| 純利益 | Green（#59A14F） |
| 総資産 | Purple（#B07AA1） |
| 負債合計 | Red（#E15759） |

### 5. グラフ（グループ棒グラフ）

| 項目 | 内容 |
|------|------|
| グラフ種別 | グループ棒グラフ（Recharts `BarChart`） |
| 横軸 | 企業の会計年度に合わせた表記（例：リクルートは「2023年3月期」）。表示期間選択に連動 |
| 縦軸 | 金額（単位：億円）。ON の全指標を同一の縦軸スケールで表示する（スケール差は許容） |
| 表示データ | トグルで ON にしている指標をすべて同一グラフに表示 |
| 各棒の色 | 指標ごとのカラーリングに従う |
| 凡例 | グラフ下部に各指標のカラーとラベルを表示（ONの指標のみ表示） |

### 6. ツールチップ

| 項目 | 内容 |
|------|------|
| 表示タイミング | グラフの棒にカーソルを当てたとき |
| 表示内容 | ON になっている全指標の「{指標名}：{金額}」を縦に並べて表示 |
| 金額の表示形式 | 1兆円以上は「X.X兆円」、1兆円未満は「X,XXX億円」と表示する |
| データなしの場合 | 該当指標の行に「データなし」と表示する（アプリをクラッシュさせない） |

## 画面状態

| 状態 | 説明 | 表示内容 |
|------|------|----------|
| ローディング | API 呼び出し中 | スピナー＋「データを読み込んでいます...」 |
| 正常 | データ取得成功 | グラフ＋指標トグル＋期間選択 |
| 全指標OFF | トグルを全部OFFにした | グラフエリアに「指標を1つ以上選択してください」を表示 |
| データなし | 全指標・全期間の値が NULL | グラフエリアに「表示できるデータがありません」を表示 |
| エラー | API エラー | 「データの取得に失敗しました。しばらくしてから再度お試しください。」 |

## 使用 API

| API | 用途 |
|-----|------|
| `GET /api/companies/{code}/financials` | 財務データ（全指標・全期分）を一括取得 |
