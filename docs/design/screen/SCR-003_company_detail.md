# SCR-003 企業詳細画面

> 画面項目とAPIレスポンスの対応は [items/SCR-003_items.md](items/SCR-003_items.md) を参照。

## 基本情報

| 項目 | 内容 |
|------|------|
| 画面ID | SCR-003 |
| 画面名 | 企業詳細画面 |
| パス | `/companies/:code` |
| 呼び出し元 | SCR-002（企業一覧画面）の企業カードクリック |

サイクル2で期間選択方式の変更（FR-12：3期/5期/10期プリセット→開始年度〜終了年度）と、
キャッシュフロー計算書の表示（FR-13）を追加した。

## 機能定義

### 1. 戻るリンク

| 項目 | 内容 |
|------|------|
| 表示条件 | 常時表示 |
| 押下時の処理 | SCR-002（企業一覧画面）へ遷移 |

### 2. 企業ヘッダー

| 項目 | 内容 |
|------|------|
| 表示内容 | 企業名・会計基準 |
| データ取得元 | `GET /api/companies/{code}/financials` のレスポンスから取得 |

### 3. 表示期間選択（サイクル2で変更）

| 項目 | 内容 |
|------|------|
| 選択方式 | 開始年度・終了年度をそれぞれセレクタで指定（サイクル1の3期/5期/10期プリセットを廃止） |
| 選択肢の範囲 | その企業についてDBに保存済みの年度のみを選択肢に出す |
| 初期選択 | DBに保存済みの最も古い年度〜最も新しい年度（＝全期間表示） |
| 選択時の処理 | `GET /api/companies/{code}/financials?from_year=&to_year=`・
  `GET /api/companies/{code}/cashflow?from_year=&to_year=`を再呼び出しする
  （サイクル1はAPI再呼び出しなしのフロント内スライスだったが、サイクル2は
  年度範囲が保存期間全体にわたりうる＝データ量が変わるため、都度サーバーに問い合わせる方式に変更） |
| データ不足時の挙動 | 指定範囲内にデータが存在しない年度は「データなし」として扱う（FR-03のエラー方針） |

### 4. 指標トグルボタン

| 項目 | 内容 |
|------|------|
| 表示内容 | 売上高 / 営業利益 / 純利益 / 総資産 / 負債合計 |
| 操作形式 | チェックボックス形式（複数同時選択可） |
| 初期状態 | 全指標をON |
| 選択時の処理 | ON の指標のみグラフに表示する（API 再呼び出しなし） |
| 表示形式 | ボタン内に指標名とその指標のカラーライン（━）を表示し、グラフの色と対応させる |

#### 指標のカラーリング

| 指標 | カラー |
|------|--------|
| 売上高 | Blue（#4E79A7） |
| 営業利益 | Orange（#F28E2B） |
| 純利益 | Green（#59A14F） |
| 総資産 | Purple（#B07AA1） |
| 負債合計 | Red（#E15759） |

### 5. グラフ（グループ棒グラフ）

| 項目 | 内容 |
|------|------|
| グラフ種別 | グループ棒グラフ（Recharts `BarChart`） |
| 横軸 | 企業の会計年度に合わせた表記（例：リクルートは「2023年3月期」）。表示期間選択に連動 |
| 縦軸 | 金額（単位：億円）。ON の全指標を同一の縦軸スケールで表示する（スケール差は許容） |
| 表示データ | トグルで ON にしている指標をすべて同一グラフに表示 |
| 各棒の色 | 指標ごとのカラーリングに従う |
| 凡例 | グラフ下部に各指標のカラーとラベルを表示（ONの指標のみ表示） |

### 6. ツールチップ

| 項目 | 内容 |
|------|------|
| 表示タイミング | グラフの棒にカーソルを当てたとき |
| 表示内容 | ON になっている全指標の「{指標名}：{金額}」を縦に並べて表示 |
| 金額の表示形式 | 1兆円以上は「X.X兆円」、1兆円未満は「X,XXX億円」と表示する |
| データなしの場合 | 該当指標の行に「データなし」と表示する（アプリをクラッシュさせない） |

### 7. キャッシュフロー計算書（サイクル2新規、FR-13）

| 項目 | 内容 |
|------|------|
| 表示位置 | グラフの下（別セクション） |
| 表示形式 | グラフ（グループ棒グラフ）＋表の両方を表示する（2026-07-21、ユーザーフィードバックによりグラフを追加）。グラフは営業CF/投資CF/財務CFの3系列、横軸＝年度。表は行＝営業CF/投資CF/財務CFの3行、列＝表示期間選択で指定した年度分 |
| データ取得元 | `GET /api/companies/{code}/cashflow?from_year=&to_year=` |
| 金額の表示形式 | グラフのツールチップと同じ形式（兆円/億円）。グラフの縦軸は財務指標グラフ（機能5）と同じく億円単位 |
| 表示期間との連動 | 表示期間選択（機能3）と連動し、財務指標グラフと同じ年度範囲を表示する |
| データなしの場合 | 表側は該当セルに「データなし」と表示する。グラフ側は`null`の期の棒を描画しない |
| 備考 | NCF（ネットキャッシュフロー）や営業CF・NCF倍率などの補助指標・折れ線は今回のスコープに含めない（ユーザー確認済み、必要になれば別途検討） |

### 8. 財務分析指標セクション（サイクル3新規、FR-23〜26）

| 項目 | 内容 |
|------|------|
| 表示位置 | キャッシュフロー計算書（機能7）の下、別セクション |
| 表示形式 | 表形式。行＝ROE・自己資本比率・EPS・PER・配当性向・ROA・総資産回転率・売上高営業利益率・売上高純利益率・流動比率・固定比率・棚卸資産回転率、列＝表示期間選択で指定した年度分（グラフ化はしない。単位が%・回・円など指標ごとに異なるため、今回は表形式のみとする） |
| データ取得元 | `GET /api/companies/{code}/ratios?from_year=&to_year=` |
| 数値の表示形式 | 比率（ROE・自己資本比率・売上高営業利益率・売上高純利益率）は%表示、回転率は「X.X回」、EPS/PERは小数第2位までの数値表示 |
| 表示期間との連動 | 表示期間選択（機能3）と連動し、財務指標グラフ・CF表と同じ年度範囲を表示する |
| データなしの場合 | 該当セルに「データなし」と表示する（US GAAP企業は現時点で全指標がデータなしになる想定） |
| 開示値・計算値の優先順位 | 自己資本比率等、EDINET開示値と計算値の両方が存在しうる指標は、開示値を優先し、なければ計算値で補完する（1セルにつき1つの値のみ表示） |

## 画面状態

| 状態 | 説明 | 表示内容 |
|------|------|----------|
| ローディング | API 呼び出し中 | スピナー＋「データを読み込んでいます...」 |
| 正常 | データ取得成功 | グラフ＋指標トグル＋期間選択＋キャッシュフロー表 |
| 全指標OFF | トグルを全部OFFにした | グラフエリアに「指標を1つ以上選択してください」を表示（キャッシュフロー表は指標トグルと独立して常に表示） |
| データなし | 全指標・全期間の値が NULL | グラフエリアに「表示できるデータがありません」を表示 |
| エラー | API エラー | 「データの取得に失敗しました。しばらくしてから再度お試しください。」 |

## 使用 API

| API | 用途 |
|-----|------|
| `GET /api/companies/{code}/financials` | 財務データ（5指標）を取得 |
| `GET /api/companies/{code}/cashflow` | キャッシュフロー（3項目）を取得 |
| `GET /api/companies/{code}/ratios` | 財務分析指標（ROE・流動比率等）を取得（サイクル3新規） |
