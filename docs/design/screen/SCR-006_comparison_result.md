# SCR-006 比較結果画面

> 画面項目とAPIレスポンスの対応は [items/SCR-006_items.md](items/SCR-006_items.md) を参照。

## 基本情報

| 項目 | 内容 |
|------|------|
| 画面ID | SCR-006 |
| 画面名 | 比較結果画面 |
| パス | `/compare/result?codes=` |
| 呼び出し元 | SCR-005（比較企業選択画面）の「比較する」ボタン |
| 関連サイクル | サイクル15 FR-66（UC-1-2-1：同業他社と指標を横並び比較する） |

`codes`（カンマ区切りの証券コード）で指定された企業について、SCR-003（企業詳細
画面）と同じカテゴリ構成で、企業別の棒グラフ・表を表示する。各企業は最新期
（`period_end`が最も新しいレコード）の値を使う。

データ取得は、`codes`の企業コードそれぞれについて既存のAPI-COM-002・API-COM-003・
API-COM-005を並行して呼び出す方式とし、新規APIは作らない。比較対象企業数は
SCR-005で上限を設けていないが、実運用では数社〜十数社程度を想定しており、
既存APIの並行呼び出しで十分と判断した。

## 機能定義

### 1. 戻るリンク

| 項目 | 内容 |
|------|------|
| 表示内容 | 「← 比較する企業を選び直す」 |
| 押下時の処理 | SCR-005（比較企業選択画面）へ遷移 |

### 2. 見出し

| 項目 | 内容 |
|------|------|
| 表示内容 | 「比較結果（{企業数}社）」 |

### 3. 貸借対照表（B/S）セクション

| 項目 | 内容 |
|------|------|
| 対象指標 | 総資産・負債・自己資本（純資産）の3指標（SCR-003と同じ） |
| 表示形式 | グループ棒グラフ（x軸＝企業名、系列＝3指標）＋表（行＝指標、列＝企業） |
| データ取得元 | `codes`の各企業について`GET /api/companies/{code}/financials`
  （年度指定なし、全期間取得のうえ最新期のレコードのみ使う） |
| 金額の整形ルール | SCR-003機能5と同じ（1兆円以上「X.X兆円」、それ未満「X,XXX億円」） |

### 4. 損益計算書（P/L）セクション

| 項目 | 内容 |
|------|------|
| 対象指標 | 売上高・営業利益・経常利益・純利益の4指標 |
| 表示形式 | 機能3と同じ仕組み |

### 5. キャッシュフロー計算書セクション

| 項目 | 内容 |
|------|------|
| 対象指標 | 営業CF・投資CF・財務CFの3指標 |
| データ取得元 | `codes`の各企業について`GET /api/companies/{code}/cashflow` |
| 表示形式 | 機能3と同じ仕組み |

### 6. 財務分析指標セクション

| 項目 | 内容 |
|------|------|
| カテゴリ構成 | 収益性・効率性・安全性・投資指標の4カテゴリ（SCR-003と同じ分類） |
| データ取得元 | `codes`の各企業について`GET /api/companies/{code}/ratios` |
| 表示形式 | カテゴリごとにグループ棒グラフ（x軸＝企業名、系列＝カテゴリ内の各指標）＋表 |
| 数値の表示形式 | SCR-003と同じ（比率は%、回転率は「X.X回」、EPS/PERは小数第2位まで） |

### 7. 企業軸ラベルのクリック

| 項目 | 内容 |
|------|------|
| 対象 | 各グラフのx軸ラベル（企業名）・各表の列見出し（企業名） |
| 押下時の処理 | SCR-003（企業詳細画面）へ遷移（パス：`/companies/{code}`） |

### 8. 一部企業の取得失敗

| 項目 | 内容 |
|------|------|
| 挙動 | 失敗した企業をグラフ・表の対象から除外し、画面上部に
  「一部の企業のデータ取得に失敗しました（{企業名}）」を表示する。
  他の企業の表示は継続する |

## 画面状態

| 状態 | 説明 | 表示内容 |
|------|------|----------|
| ローディング | 各企業のAPI呼び出し中 | 「データを読み込んでいます...」 |
| 正常 | `codes`で指定した企業のうち1件以上取得成功 | 全セクション表示（機能3〜6） |
| 空 | `codes`が空、または該当企業が1件もない | 「比較する企業が選択されていません」＋
  SCR-005へのリンク |
| 一部失敗 | 一部企業のAPI呼び出しが失敗 | 機能8の警告表示＋成功した企業のみで表示継続 |

## 使用 API

| API | 用途 |
|-----|------|
| `GET /api/companies/{code}/financials` | 各企業のB/S・P/Lデータを取得 |
| `GET /api/companies/{code}/cashflow` | 各企業のキャッシュフローを取得 |
| `GET /api/companies/{code}/ratios` | 各企業の財務分析指標を取得 |
