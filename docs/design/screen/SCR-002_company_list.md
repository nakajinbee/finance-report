**サイクル14での変更**：全上場企業（約3,800社）の一括登録（サイクル6）以降、
初期表示時に全件をカード表示すると閲覧しづらいため、「検索または業種絞り込みを
指定した場合のみカード一覧を表示する」方式に変更した。あわせて業種絞り込み・
並び順の指定を追加した（FR-61〜64）。

## 機能定義

### 1. 検索バー

| 項目 | 内容 |
|------|------|
| 表示条件 | 常時表示 |
| 入力形式 | テキスト入力（部分一致） |
| 検索対象 | 企業名・証券コード |
| 検索タイミング | 入力のたびにリアルタイムでリストを絞り込む（API 再呼び出しなし・フロントで絞り込み） |
| 大文字小文字 | 区別しない |
| ひらがな・カタカナ | 区別しない（「りくるーと」で「リクルート」を検索可能） |
| 初期状態 | 空欄 |

### 2. 業種絞り込み（サイクル14で追加）

| 項目 | 内容 |
|------|------|
| 表示条件 | 常時表示（検索バーの隣） |
| 入力形式 | セレクトボックス |
| 選択肢 | 取得済み企業一覧に含まれる`sector`の重複なしリスト（五十音順）＋「すべて」（先頭） |
| 除外対象 | `sector`が`null`（データ未取得）の企業は選択肢の生成対象から除く |
| 絞り込みの適用 | キーワード検索とAND条件で適用する |
| 初期状態 | 「すべて」 |

### 3. 並び順（サイクル14で追加）

| 項目 | 内容 |
|------|------|
| 表示条件 | 常時表示（業種絞り込みの隣） |
| 入力形式 | セレクトボックス |
| 選択肢 | 証券コード昇順（初期状態）／証券コード降順／企業名昇順（あいうえお順）／企業名降順 |
| 適用対象 | キーワード検索・業種絞り込み後の結果に対して適用 |

### 4. 件数表示

| 項目 | 内容 |
|------|------|
| 表示条件 | キーワードが空でない、または業種が「すべて」以外（＝絞り込み条件がある）場合のみ |
| 表示内容 | 「{全件数}件中 {表示件数}件表示」 |
| 更新タイミング | 検索・業種絞り込みに連動して更新 |

### 5. 企業カード一覧

| 項目 | 内容 |
|------|------|
| 表示条件 | 絞り込み条件（キーワードまたは業種）があり、かつ結果が1件以上ある場合。
  **絞り込み条件がない場合は表示しない**（サイクル14変更点） |
| ソート順 | 「2. 並び順」の選択に従う |
| クリック時の処理 | SCR-003（企業詳細画面）へ遷移（パス：`/companies/{code}`） |
| 使用 API | `GET /api/companies`（初回のみ。以降はフロントで絞り込み・ソート） |

#### 企業カードの表示項目

| 項目 | 内容 | 表示例 |
|------|------|--------|
| 企業名 | 正式名称 | リクルートホールディングス |
| 証券コード | 4桁 | 6098 |
| 業種 | EDINET の業種区分 | サービス業 |
| 会計基準 | IFRS / J-GAAP / US GAAP | IFRS |

### 6. データ追加取得ボタン

| 項目 | 内容 |
|------|------|
| 表示条件 | 常時表示 |
| 押下時の処理 | SCR-001（ダウンロード画面）へ遷移 |

## 画面状態

| 状態 | 説明 | 表示内容 |
|------|------|----------|
| ローディング | API 呼び出し中 | スピナー＋「読み込み中...」 |
| 正常（データなし） | 保存済み企業が0件 | 「データがありません。まずデータを取得してください。」＋ダウンロード画面へのリンク |
| 未絞り込み（サイクル14で追加） | 保存済み企業は1件以上あるが、キーワード未入力・業種未選択 | 検索バー・業種絞り込み・並び順は表示するが、案内メッセージ
  「企業名・証券コードで検索するか、業種を選択してください」を表示し、カード一覧・件数表示は出さない |
| 正常（絞り込み結果あり） | 絞り込み条件があり結果が1件以上 | 検索バー・業種絞り込み・並び順・件数表示・企業カード一覧 |
| 絞り込み結果0件 | 絞り込み条件があるが結果が0件 | 「"{キーワード}" に一致する企業が見つかりませんでした」 |
| エラー | API エラー | 「データの取得に失敗しました。しばらくしてから再度お試しください。」 |
