# IDEA-02：企業一覧ページ：検索結果のみ表示する形式に変更

## 概要
企業一覧ページを、検索した企業だけが表示される形式に変える。

## 詳細
現状の挙動（全件表示＋クライアント側絞り込み）を、検索前は一覧を出さない、または検索して
初めて結果が表示される形式に変更する。IDEA-01（DB一括化）で企業数が大幅に増える想定なら、
全件表示は現実的でなくなるため、この変更の必要性が高まる。

## いつ反映するか
未定（IDEA-01の状況次第で優先度が上がる可能性あり）

## 優先度
未定

## メモ
IDEA-01が前提になるかは要検討。全件投入前でも、UXとして検索必須にする価値があるかは
別途判断する。
