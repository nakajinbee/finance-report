# UC-1-1 特定企業の深掘り調査

[ユースケース一覧](../../use_cases.md)のUC-1-1に対応する詳細。

## 小項目

| ID | 内容 | 状態 |
|---|---|---|
| UC-1-1-1 | 財務諸表（B/S・P/L・CF）を確認する | 実現済み |
| UC-1-1-2 | 財務分析指標を確認する | 実現済み |
| UC-1-1-3 | 事業内容・リスク情報等の定性情報を確認する | 実現済み |

## 画面遷移

新規画面・新規遷移は不要。既存のSCR-002（企業一覧）→SCR-003（企業詳細）という
画面フローの中で完結する。UC-1-1-1・UC-1-1-2はSCR-003内で対応済み。UC-1-1-3
（定性情報）は、SCR-003に新しいセクションを追加する形で対応する（画面遷移自体の
追加はなし）。
